Cette modélisation est une version antérieure à « Cueilleur-Artisan-Agriculteur ».

On peut remarquer que dans ce modèle, les agriculteurs n’apparaissent pas de façon émergente. 
Dans chaque groupe, un chef est élu et régule le besoin de baies dans le groupe en augmentant le nombre d’agriculteur lorsque l’insatisfaction augmente.

Remarques:
-Il n’y a pas de système de mort avec une « EsperanceDeVie ».
